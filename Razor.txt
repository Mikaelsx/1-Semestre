Razor é uma linguagem de marcação útilizada para incorporar o código.

@ - propriedade razor

Razor oferece suporte a C# e usa o @símbolo para fazer a transição de HTML para C#.

Para escapar de um @símbolo na marcação Razor, use um segundo @símbolo:

<p>@@Username</p>
